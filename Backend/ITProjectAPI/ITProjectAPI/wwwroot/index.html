Welkom bij de API van de The ArchITects

Ontwikkeld door The ArchITects™

Wat is dit?

Deze API haalt de belangrijkste financiële data van alle Belgische bedrijven op via de (test) API van de Nationale Bank van België. Buiten het gewoon teruggeven van de individuele data, wordt er ook een vergelijking van twee bedrijven voorzien. e opzoeking gebeurt via het KB)-nummer.
Alle mogelijke routes en datasoorten worden hieronder meegedeeld.

Wie zijn we?

Deze API wordt u gebracht als deel van het vak IT Project, opleiding graduaat Programmmeren van de AP Hogeschool. Ontwikkeld door "The ArchITects", een gedreven team van dynamische developers genaamd Valerie, Sam en Qin.

Dataformaat

Alle responses die deze API voorziet, worden aangeleverd in JSON-formaat.

Routes

Alle routes worden voorafgegaan door de string https://nbb-architects-api.azurewebsites.net/home.

Endpoint Response
/gegevens/{kbonummer} Adresgegevens van het bedrijf
/accountingdata/{kbonummer} Financiële gegevens van het bedrijf
  • deposit date
  • eigen vermogen
  • schulden
  • bedrijfswinst
