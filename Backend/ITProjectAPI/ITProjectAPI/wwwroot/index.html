Welkom bij de API van The ArchITects

Ontwikkeld door The ArchITects™

Wat is dit?

Deze API haalt de belangrijkste financiële data van alle Belgische bedrijven op via de (test) API van de Nationale Bank van België. Buiten het gewoon teruggeven van de individuele data, wordt er ook een vergelijking van twee bedrijven voorzien. De opzoeking gebeurt via het KBO-nummer.
Alle mogelijke routes en datasoorten worden hieronder meegedeeld.

Wie zijn we?

Deze API wordt u gebracht als deel van het vak IT Project, opleiding graduaat Programmmeren van de AP Hogeschool. Ontwikkeld door "The ArchITects", een gedreven team van dynamische developers genaamd Valérie, Sam en Qin.

Dataformaat

Alle responses die deze API voorziet, worden aangeleverd in JSON-formaat.

Routes

Alle routes worden voorafgegaan door de URI https://nbb-architects-api.azurewebsites.net/.

Endpoint Response
home/gegevens/{kbonummer} Adresgegevens van het bedrijf
home/accountingdata/{kbonummer} Financiële gegevens van het bedrijf
  • datum van neerlegging
  • eigen vermogen
  • schulden
  • bedrijfswinst
search/alldata Adres- en financiële gegevens van de vier laatste databank entries
search/searchquery/{kbonummer/bedrijfsnaam} Adres- en financiële gegevens van het opgezochte bedrijf
